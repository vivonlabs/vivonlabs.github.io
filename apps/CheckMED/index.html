📱 CheckMED

약속을 지키는 복약 알림 앱
병원/약국에서 받은 약을 정확한 시간에 복용할 수 있도록, 꺼져도·재부팅해도 신뢰할 수 있는 알림과 기록을 제공합니다.

🔍 앱 개요

  • 앱 목적: 정해진 시간에 약을 놓치지 않도록 “플랜-알림-기록”을 한 번에
  • 대상 사용자: 정기 복용이 필요한 분, 장기 복약 스케줄을 관리해야 하는 분
  • 프라이버시: 회원가입/서버 없이 기기 내부에만 저장(암호화)

⚙️ 주요 기능

  • 복약 플랜 생성: 병원명/약국명/복용 기간/시간대만 입력하면 전체 복용 일정을 자동으로 생성합니다. (최대 90일 이하 관리)
  • 정확한 시간 알림: 앱이 백그라운드이거나 꺼진 상태에서도 약속한 시간에 알림이 울리도록 설계했습니다.
  • 복용 기록 관리: 시간대별로 복용 완료/미복용을 기록하고, 플랜별 진행 현황(완료율, 남은 횟수)을 확인할 수 있습니다.

✅ CheckMED가 신뢰할 수 있는 이유

  • 라이프사이클 대응: Foreground/Background/종료(Terminated) 상황에서도 알림 신뢰성에 집중
  • 시간 정확도: 목표 시각 기반으로 스케줄을 관리해 알림 지연/표류를 최소화
  • 로컬 우선: 서버/클라우드가 없어 개인정보가 외부로 나갈 경로를 차단

🔐 보안 & 프라이버시

  • 데이터는 기기 내부에만 저장되며 외부 서버로 전송되지 않습니다.
  • 민감 정보는 AES-256 암호화로 보호됩니다.
  • 암호화 키는 Android Keystore / iOS Keychain 기반으로 안전하게 관리됩니다.

🧭 사용 방법(간단한 흐름)

  1. 플랜 생성: “+” 버튼 → 병원/약국/기간 입력 → 시간 설정 → 저장
  2. 알림 수신: 설정한 시간에 복약 알림 수신
  3. 복용 확인: 복용완료/미복용으로 기록
    • 30분 이내: 대기중(녹색)
    • 1시간 이내: 대기중(주황색)
    • 1시간 이상: 시간 경과 → 미복용으로 전환
  4. 진행 확인: 플랜 목록에서 완료율/남은 횟수 확인
